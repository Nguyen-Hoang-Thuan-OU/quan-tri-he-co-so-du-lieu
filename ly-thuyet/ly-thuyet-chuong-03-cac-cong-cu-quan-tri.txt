I. SQL Server Management Studio:

Vì SQL Server không có giao diện, muốn quản trị thì phải sử dụng dòng lệnh, nên Microsoft phát hành thêm công cụ đồ hoạ để quản trị bên cạnh nhiều công cụ đồ hoạ giúp quản trị khác.
	-> Là một công cụ rời để quản trị SQL Server thay vì sử dụng dòng lệnh.

* Studio: trọn gói - đa năng, gồm nhiều công cụ, được tổ chức theo dạng giao diện Medium-dependent interface (MDI) với cửa sổ cha và nhiều cửa sổ con, cho phép float, dock hoặc tổ chức thành dạng tab.

=> Tất cả các tác vụ đều có thể giải quyết bằng code, công cụ đồ hoạ chỉ giúp việc quản trị trở nên dễ dàng và nhanh chóng hơn (có thể sẽ thiếu chức năng).

--------------------------------------------------

SQL Server Management Studio thay thế hoàn toàn cho Enterprise Manager và Query Analyzer.

--------------------

** Các cửa sổ (con) chức năng:
	01. Object Explorer: cửa sổ thường được sử dụng nhất. Sử dụng menu ngữ cảnh nên khi nhấn chuột phải, tuỳ vào loại tập tin mà sẽ có những chức năng tương ứng.

	02. Code Editor: nơi để viết code, ví dụ: New Query.

	03. Solution Explorer: nơi quản lý các dự án lập trình, quản lý Solution và Project giống như Visual Studio, một Solution sẽ gồm nhiều Project. Trong VS, mỗi Project có thể được viết bằng nhiều ngôn ngữ lập trình khác nhau (C, C#, VB,...) và các Project khác nhau khi build sẽ build chung trong một Solution.

	04. Properties Window: cửa sổ thuộc tính của những đối tượng đang được chọn.

	05. Registered Servers: cửa sổ dùng để đăng ký Server lên một máy khác trong cùng mạng LAN để thực hiện một số thao tác quản trị cơ bản từ xa. Khi quản trị Server, rất ít khi được tiếp xúc với máy Server, Registered Servers sẽ giúp đăng ký máy Server lên máy tính cá nhân, cho phép điều khiển máy Server từ xa bằng cách đăng nhập qua mạng LAN với quyền người dùng tối thượng và thực hiện một số thao tác ở mức cơ bản.

	06. Bookmark Window: thường dành cho lập trình viên CSDL, giúp đánh dấu lại đoạn code đang viết.

	07. Toolbox: hộp đồ nghề dành riêng cho dân quản trị CSDL.

	08. Object Explorer Details (Summary Window): cho biết chi tiết những phần được chọn trong Object Explorer.

	09. Web Browser: trình duyệt web được nhúng sẵn. Trong những sản phẩm đời mới của Microsoft, file hướng dẫn (help) được ghép từ những file HTML thường có đuôi .chm, cần HTML Viewer để đọc hướng dẫn dạng website.

	10. Template Explorer: thư viện mã nguồn có sẵn dành cho lập trình viên, giúp lập trình viên không cần phải nhớ code mà chỉ cần sử dụng những mà nguồn đã được viết sẵn.

	11. Toolbars: thanh công cụ, có thể cầm và kéo đi bất cứ đâu trong phạm vi cho phép.

	12. Custom Toolbar: cho phép thêm một số nút vào thanh công cụ để cá nhân hoá thanh công cụ.

	13. Database Diagram Toolbar: thanh công cụ được dùng để làm việc với Database Diagram.

	14. Help Toolbar: công cụ trợ giúp.

	15. Query Designer Toolbar: công cụ thiết kế truy vấn (kéo bảng, liên kết các bảng, thêm cột,...).

	16. Source Control Toolbar: công cụ quản lý mã nguồn, có chức năng sao lưu, né, phân quyền/chia sẻ file,...

	17. SQL Editor Toolbar

	18. SQL Mobile Editor Toolbar

	19. SQL Server Analysis Services Editor Toolbar

	20. Standard Toolbar

	21. Table Designer Toolbar

	22. Text Editor Toolbar

	23. View Designer Toolbar

	24. SQL Server Management Studio Configuration: chỉnh cấu hình/tuỳ chọn cho Management Studio, lưu lại những tuỳ chỉnh này để tái sử dụng cho những lần sau thay vì phải cấu hình lại từ đầu. Lưu những tuỳ chỉnh của người dùng vào một tập tin riêng, quét tập tin và gán lại vào biến thông số khi cần.
		1. Environment
		2. Source Control
		3. Text Editor
		4. Query Execution
		5. Query Results
		6. Designers

========================================================================

II. Log File Viewer:

Xem nội dung của log file, là một phần mềm rời, vừa sử dụng được cho hệ điều hành Windows và vừa sử dụng được cho SQL Server. Có thể mở từ Management Studio và có kiểm tra quyền đăng nhập trước khi cho phép xem.

========================================================================

III. SQL Server Business Intelligence Development Studio:

SQL Server Business Intelligence Development Studio (BI Studio) thật sự chính là 1 phiên bản thu nhỏ của Visual Studio 2005, phần dịch vụ kèm thêm để lập trình CSDL.

3 phần dịch vụ chạy ngầm kèm thêm của gói BI (business intelligence package):
	1. SQL Server Integration Services. (Tích hợp)
	2. SQL Server Analysis Services. (Phân tích)
	3. SQL Server Reporting Services. (Báo cáo)

========================================================================

IV. SQL Server Profiler: phần mềm độc lập với Management Studio, giúp tạo trace/track và theo dõi/truy vết dữ liệu, tiến trình (process), những đối tượng tình nghi có nguy cơ gậy hại đến hệ thống khiến CPU cao.

========================================================================

V. Database Engine Tuning Advisor: phần mềm đưa ra lời khuyên để tinh chỉnh việc quản trị CSDL, thường liên quan đến dạng chuẩn, dư/thừa dữ liệu,... Ví dụ: cột nào có thao tác tìm kiếm rất nhiều nhưng chưa có index, 2 bảng DonHang và LoaiHang có 1 cột trùng tên nhưng không phải là khoá ngoại/khoá chính tham khảo lẫn nhau, 2 bảng không liên quan đến nhau nhưng cùng tên trường (field).

========================================================================

VI. SQL Server Configuration Manager: là công cụ của Server, giúp tinh chỉnh những thông số quan trọng của Server, chẳng hạn như giao thức kết nối, alias,...

Có thể sử dụng ở máy Server và Client (SQL Native Client). Mặc định sẽ được cài khi cài Server, còn Client nếu muốn sử dụng thì phải cài thêm.

Management Tool là một phần mềm phía Client chạy kiến trúc 32-bit, nên khi cần chỉnh thì sẽ vào SQL Native Client Configuration (32bit) để chỉnh.

========================================================================

VII. Reporting Services Configuration Manager: chỉnh các thông số liên quan đến chức năng báo cáo (1 trong 4 dịch vụ cài thêm).

========================================================================

VIII. Command-Line Tools: quản trị CSDL bằng cửa sổ dòng lệnh.
